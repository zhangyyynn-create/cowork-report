产品动向

企业 Agent

OpenAI 发布 ChatGPT Work，跨应用执行任务

2026-07-09 · BNN Bloomberg · BNN Bloomberg

OpenAI 于 7 月 9 日推出 ChatGPT Work，这是一个嵌入 ChatGPT 的 Agent，能够跨不同应用和文件执行任务，例如在 Google Sheets 中更新数据、在 Slack 中发送消息、在 Notion 中创建文档等。该产品面向企业用户，旨在自动化重复性工作流。据 BNN Bloomberg 报道，ChatGPT Work 目前处于早期访问阶段，定价尚未公开，但预计将采用按任务或按席位收费模式。

证据拼图/多源验证

BNN Bloomberg 的报道是主要来源，但 OpenAI 官方尚未发布详细公告。报道中引用了 OpenAI 发言人的声明，确认了产品名称和核心功能。由于缺乏官方产品页面或 changelog，该信息需结合后续官方文档进一步验证。不过，Bloomberg 作为权威商业媒体，其报道可信度较高。

能力维度影响

ChatGPT Work 直接切入“跨应用执行”和“企业知识/文件/IM/邮件接入”两个关键维度。它能够连接 Slack、Google Sheets、Notion 等常见办公工具，意味着用户可以在一个对话界面中调度多个 SaaS 应用，减少上下文切换。这类似于 Kimi Work 的跨应用能力，但 OpenAI 的品牌效应和现有用户基础可能加速企业采纳。

简评：ChatGPT Work 是 OpenAI 在企业级 AI cowork 领域的重要布局。其跨应用执行能力直接对标 Kimi Work 和微软 Copilot，但 OpenAI 的优势在于 ChatGPT 已有的庞大用户群和强大的模型能力。然而，企业客户对数据安全和权限管理要求极高，OpenAI 需要展示其合规性和企业级支持。此外，定价策略将决定其市场渗透速度——如果按任务收费过高，可能仅适合高价值场景；若按席位低价切入，则可能快速抢占中小企业市场。建议关注后续官方定价和实际任务成功率。
垂直 Agent

Akeneo 发布 Agentic 产品体验平台，多 Agent 协作

2026-07-08 · TMCnet · TMCnet

Akeneo 在 Summer Release 中将其产品云从记录系统升级为 Agentic 平台，引入多 Agent 协作机制。这些 Agent 能够自主协调完成产品信息管理任务，如数据 enrichment、分类、翻译和合规检查。Akeneo 声称这是首个真正 Agentic 的产品体验平台，旨在减少人工操作，提高产品数据质量。

证据拼图/多源验证

TMCnet 的报道是主要来源，但 Akeneo 官方博客和产品页面也同步更新了相关信息。Akeneo 作为产品信息管理（PIM）领域的知名厂商，其产品升级有明确的行业背景。多 Agent 协作在 PIM 场景中具有实际价值，例如一个 Agent 负责抓取供应商数据，另一个进行标准化，第三个检查合规性。这种垂直领域的 Agent 协作模式值得其他行业借鉴。

对 cowork 产品的启发

Akeneo 的案例展示了 AI cowork 在垂直场景中的落地方式：不是通用助手，而是由多个专业 Agent 组成的“团队”协同工作。这提示通用 cowork 产品（如 ChatGPT Work）可能需要提供 Agent 编排能力，让用户定义任务分解和协作流程。同时，Akeneo 的 Agent 专注于产品数据，说明领域知识是 Agent 有效性的关键。

简评：Akeneo 的 Agentic 平台是垂直领域 AI cowork 的典范。它没有试图做一个通用助手，而是深入产品信息管理这一具体场景，用多 Agent 协作解决数据质量痛点。这种策略降低了通用 Agent 的“幻觉”风险，因为每个 Agent 的任务边界清晰。对于通用 cowork 产品，Akeneo 的启发是：提供可配置的 Agent 模板和任务编排工具，让企业能快速构建自己的 Agent 团队。不过，Akeneo 的 Agent 目前仅限其平台内部，未来若开放 API 与其他系统集成，将更具竞争力。

关键判断

1. OpenAI 正式进入企业 Agent 市场，ChatGPT Work 跨应用执行能力将加剧与微软、Kimi 的竞争。

2. 垂直领域多 Agent 协作模式（如 Akeneo）证明 AI cowork 在特定场景下比通用助手更有效。

3. 跨应用执行成为 AI cowork 产品的标配能力，但数据安全和权限管理仍是企业采纳的关键障碍。

4. 定价模式（按任务 vs 按席位）将影响 AI cowork 产品的市场渗透策略。

5. 本周其他主要产品（Kimi、Trae、Qoder）无重大更新，行业焦点转向 OpenAI 的新动作。
